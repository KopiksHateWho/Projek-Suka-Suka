## 2025-05-15 - [Password Visibility UX Failure]
**Learning:** Even simple UX improvements like a password visibility toggle can backfire if the visual implementation is not carefully integrated into the existing design. Using emojis and generic button styles often fails to match the professional "gaming" aesthetic of the site, and absolute positioning must be used precisely to place the toggle *inside* the input field to meet user expectations.
**Action:** Always use SVGs for icon-only buttons to ensure consistency and use precise CSS (e.g., `right-4 top-1/2 -translate-y-1/2`) to place icons within input boundaries.
